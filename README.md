# 주간 계획표 앱 (Schedule App)

요일별 시간 블록으로 일주일 계획을 짜고, **상황별 프리셋(방학·학기중·휴가 등)** 으로 저장해
한눈에 확인하는 앱입니다.

## 핵심 기능

- **블록 시간표**: 시간대별 할 일을 색상 블록으로 관리
- **프리셋 저장**: 일주일 시간표를 라벨링해 상황별로 저장/전환
- **현재 할 일 하이라이트**: 지금 시각에 해당하는 블록을 자동 강조 (`지금` 배지)
- **AI 시간표 생성** *(예정)*: 자연어로 할 일을 말하면 시간표 자동 생성
- **데스크탑/모바일 위젯** *(예정)*

## 기술 스택

| 영역 | 스택 |
|------|------|
| 웹 | React + Vite + TypeScript, Vercel 배포 |
| DB | Supabase *(2단계)* |
| 로그인 | 카카오/네이버 소셜 로그인 *(2단계)* |
| 앱 | 안드로이드 우선 *(4단계)* |

## 개발 로드맵

- **1단계 ✅** — 웹 MVP: 블록 시간표 + 프리셋(로컬 저장) + 현재 할일 하이라이트 + 편집
- **2단계 🚧 (현재)** — Supabase 연동(클라우드 저장) + 카카오 로그인. *코드 완료, 대시보드 설정만 남음 → [설정 가이드](docs/2단계-Supabase-카카오-설정.md)*. 네이버는 이후 커스텀 추가.
- **3단계** — AI 시간표 생성 (직접 SDK 사용, liteLLM 미사용)
- **4단계** — 데스크탑/모바일 위젯, 안드로이드 앱

> **로그인 없이도 동작**합니다. Supabase 키가 없으면 자동으로 로컬 전용 모드로 실행됩니다.

## 실행 방법

```bash
cd web
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 프로덕션 빌드 (dist/)
```

## 폴더 구조

```
schedule_app/
├─ assets/            # 참고용 원본 HTML 시안
└─ web/               # React 웹 앱 (1단계)
   └─ src/
      ├─ components/  # UI 컴포넌트 (프리셋바, 요일탭, 카드, 편집 모달)
      ├─ data/        # 기본 프리셋 데이터
      ├─ hooks/       # 커스텀 훅 (현재 시각 갱신)
      ├─ lib/         # 순수 유틸 (시간 계산, 저장, id)
      └─ types.ts     # 공통 데이터 타입
```

## 현재 저장 방식

1단계는 **로컬 우선(local-first)** 으로 브라우저 `localStorage` 에 저장합니다.
2단계에서 `web/src/lib/storage.ts` 의 구현만 교체하면 Supabase 클라우드 동기화로 확장됩니다.
